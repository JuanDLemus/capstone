# Capstone Project - Informatics Sabana

A React-based web application for waste management and informatics capstone.

## Tech Stack
- **Framework**: React (Vite)
- **Styling**: Vanilla CSS
- **Icons**: Lucide React

## Development
```bash
npm install
npm run dev
```

## Future Roadmap
- Migration to React Native for native APK deployment.
- Integration with F-Droid for open-source distribution.
